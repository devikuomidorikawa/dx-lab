## 課題

大学事務では、教員・学生・外部機関など立場の異なる相手に日常的にメールを送信する。丁寧な文面を一から考えると1通あたり10〜15分かかることもあり、件数が多い日は大きな負担になる。特に返信メールは、受信内容を読み取ったうえで適切な回答を組み立てる必要があり、時間がかかりやすい。

## この記事が役立つケース

**導入が効果的な場合:**
- メールの送受信件数が多く、返信に時間がかかっている
- 文面を一から考えることが多く、テンプレートだけでは対応しきれない
- 丁寧な敬語や適切な表現に迷うことがある

**現状のままで十分な場合:**
- メールの件数が少なく、作成に困っていない
- 定型文のテンプレートで大半のメールをカバーできている

## 解決策

**Gmail** に搭載された **Gemini** の機能を使い、受信メールへの返信案を自動生成する。また、新規メールの作成も Gemini に指示するだけでドラフトを生成できる。追加のツールやセットアップは不要で、Gmail の画面上ですぐに使える。

## 前提条件

- Google Workspace Business Standard 以上、または Google One AI Premium プラン
- Gmail の Gemini 機能が管理者により有効化されていること

## 返信メールの作成

### 1. 返信案の自動提案

受信メールを開いて「返信」をクリックすると、メール本文の下に Gemini が提案する返信案がチップ（ボタン）として表示される。

**表示される返信案の例:**
- 「承知しました」
- 「ありがとうございます」
- 「確認して折り返します」

チップをクリックすると、その趣旨に沿った丁寧な返信文が自動生成される。内容を確認・修正して送信する。

### 2. 自分で指示して返信を作成する

提案されたチップが合わない場合は、自分で指示を出して返信を作成できる。

**手順:**

1. 受信メールを開き「返信」をクリック
2. 返信作成画面の下部にある **Gemini アイコン（ペンのマーク）** をクリック
3. 「このメールへの返信を手伝う」を選択
4. 作成したい返信の要旨を入力する

**入力例:**

```
3月15日の会議について、出席できる旨を伝えてください。
資料は事前に確認しておく旨も添えてください。
```

```
申請の件について、添付の書類に不備があるため
再提出をお願いする返信を作成してください。
丁寧な表現でお願いします。
```

```
この問い合わせに対して、担当部署は教務課であること、
教務課の連絡先（内線1234）を案内する返信を作成してください。
```

5. Gemini が返信文を生成する
6. 内容を確認し、必要に応じて修正して送信

### 3. 文体の調整

生成された返信文は、以下のオプションで調整できる。

| オプション | 効果 |
|-----------|------|
| フォーマルにする | より丁寧な敬語表現に調整 |
| カジュアルにする | 親しみのある表現に調整 |
| 短くする | 冗長な表現を削り簡潔にする |
| 詳しくする | 説明を補足して丁寧にする |

相手に応じて使い分ける。教員や外部機関向けには「フォーマル」、日常的にやり取りのある同僚には「カジュアル」が適している。

## 新規メールの作成

Gemini は返信だけでなく、新規メールの作成もサポートしている。

**手順:**

1. Gmail で「作成」をクリックし、新規メール作成画面を開く
2. 下部の **Gemini アイコン（ペンのマーク）** をクリック
3. 「メールの作成を手伝う」を選択
4. 作成したいメールの内容を指示する

**入力例（会議の案内）:**

```
教務委員会の委員宛に、次回会議の案内メールを作成してください。

日時: 3月20日（木）14:00〜15:30
場所: 本館3階 会議室A
議題: 来年度カリキュラム改定について
持ち物: 配布済みの改定案

出欠の回答を3月17日までにお願いする旨を含めてください。
```

**入力例（依頼メール）:**

```
総務課の鈴木さん宛に、会議室の予約を依頼するメールを作成してください。
4月10日の午後、20名程度が入れる部屋を希望します。
プロジェクターも使いたい旨を伝えてください。
```

**入力例（報告メール）:**

```
部長宛に、先日のオープンキャンパスの実施報告メールを作成してください。
来場者数は約300名、昨年比120%でした。
アンケート結果は概ね好評で、詳細は別途報告書にまとめる予定です。
```

5. Gemini がメール文面を生成する
6. 宛先・件名を入力し、内容を確認・修正して送信

## 活用のコツ

- **指示は具体的に** — 「お礼のメールを書いて」より「3月10日の研修会の講師を務めていただいたお礼のメールを書いて」の方が的確な文面が生成される
- **相手の立場を明示する** — 「教員宛に」「学生宛に」「外部の取引先宛に」と指示すると、適切な敬語レベルで作成される
- **含めたい情報を箇条書きで渡す** — 日時・場所・締切などの具体的な情報を箇条書きで指示すると、漏れなく文面に反映される
- **生成後に「フォーマル」「短く」などで微調整する** — 一発で完璧な文面にならなくても、調整オプションで仕上げられる

## 導入効果

| 項目 | 導入前 | 導入後 |
|------|--------|--------|
| 返信メール作成時間 | 1通あたり 5〜15 分 | 1〜3 分（確認・修正のみ） |
| 新規メール作成時間 | 1通あたり 10〜20 分 | 2〜5 分 |
| 敬語・表現への不安 | 特に新任者に多い | AI が適切な表現を提案 |

## 注意点

- 生成された文面は必ず送信前に内容を確認する。事実関係（日時、金額、固有名詞）が正確かどうかは自分で検証する
- 機密情報や個人情報を含むメールの場合、組織のセキュリティポリシーに従って利用する
- AI が生成した文面をそのまま送るのではなく、自分の言葉で微調整を加えると、より自然なコミュニケーションになる
