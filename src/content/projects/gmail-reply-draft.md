## 課題

大学事務では、教員・学生・外部機関など立場の異なる相手に日常的にメールを送信する。丁寧な文面を一から考えると1通あたり10〜15分かかることもあり、件数が多い日は大きな負担になる。また、Gemini で返信案を生成しても一般的な表現になりがちで、「自分が書いた感じ」にならず、結局大幅に書き直すことになりやすい。

## この記事が役立つケース

**導入が効果的な場合:**
- メールの送受信件数が多く、返信に時間がかかっている
- 文面を一から考えることが多く、テンプレートだけでは対応しきれない
- AI が生成する文面が自分のスタイルと合わず、毎回修正が多い

**現状のままで十分な場合:**
- メールの件数が少なく、作成に困っていない
- 定型文のテンプレートで大半のメールをカバーできている

## 解決策

**Gmail** に搭載された **Gemini** の機能を使い、受信メールへの返信案や新規メールのドラフトを生成する。さらに、**Gem**（Gemini のカスタム AI）に自分のメールスタイルを学習させておけば、Gmail のサイドパネルから自分らしい文面をワンクリックで生成できる。

## 前提条件

- Google Workspace Business Standard 以上、または Google One AI Premium プラン
- Gmail の Gemini 機能が管理者により有効化されていること

## 返信メールの作成

### 1. 返信案の自動提案

受信メールを開いて「返信」をクリックすると、メール本文の下に Gemini が提案する返信案がチップ（ボタン）として表示される。

**表示される返信案の例:**
- 「承知しました」
- 「ありがとうございます」
- 「確認して折り返します」

チップをクリックすると、その趣旨に沿った丁寧な返信文が自動生成される。内容を確認・修正して送信する。

### 2. 自分で指示して返信を作成する

提案されたチップが合わない場合は、自分で指示を出して返信を作成できる。

**手順:**

1. 受信メールを開き「返信」をクリック
2. 返信作成画面の下部にある **Gemini アイコン（ペンのマーク）** をクリック
3. 「このメールへの返信を手伝う」を選択
4. 作成したい返信の要旨を入力する

**入力例:**

```
3月15日の会議について、出席できる旨を伝えてください。
資料は事前に確認しておく旨も添えてください。
```

```
申請の件について、添付の書類に不備があるため
再提出をお願いする返信を作成してください。
丁寧な表現でお願いします。
```

```
この問い合わせに対して、担当部署は教務課であること、
教務課の連絡先（内線1234）を案内する返信を作成してください。
```

5. Gemini が返信文を生成する
6. 内容を確認し、必要に応じて修正して送信

### 3. 文体の調整

生成された返信文は、以下のオプションで調整できる。

| オプション | 効果 |
|-----------|------|
| フォーマルにする | より丁寧な敬語表現に調整 |
| カジュアルにする | 親しみのある表現に調整 |
| 短くする | 冗長な表現を削り簡潔にする |
| 詳しくする | 説明を補足して丁寧にする |

相手に応じて使い分ける。教員や外部機関向けには「フォーマル」、日常的にやり取りのある同僚には「カジュアル」が適している。

## 新規メールの作成

Gemini は返信だけでなく、新規メールの作成もサポートしている。

**手順:**

1. Gmail で「作成」をクリックし、新規メール作成画面を開く
2. 下部の **Gemini アイコン（ペンのマーク）** をクリック
3. 「メールの作成を手伝う」を選択
4. 作成したいメールの内容を指示する

**入力例（会議の案内）:**

```
教務委員会の委員宛に、次回会議の案内メールを作成してください。

日時: 3月20日（木）14:00〜15:30
場所: 本館3階 会議室A
議題: 来年度カリキュラム改定について
持ち物: 配布済みの改定案

出欠の回答を3月17日までにお願いする旨を含めてください。
```

**入力例（依頼メール）:**

```
総務課の鈴木さん宛に、会議室の予約を依頼するメールを作成してください。
4月10日の午後、20名程度が入れる部屋を希望します。
プロジェクターも使いたい旨を伝えてください。
```

5. Gemini がメール文面を生成する
6. 宛先・件名を入力し、内容を確認・修正して送信

## 活用のコツ

- **指示は具体的に** — 「お礼のメールを書いて」より「3月10日の研修会の講師を務めていただいたお礼のメールを書いて」の方が的確な文面が生成される
- **相手の立場を明示する** — 「教員宛に」「学生宛に」「外部の取引先宛に」と指示すると、適切な敬語レベルで作成される
- **含めたい情報を箇条書きで渡す** — 日時・場所・締切などの具体的な情報を箇条書きで指示すると、漏れなく文面に反映される

## Gem に自分のメールスタイルを学習させる

ここまでの方法では、Gemini が生成する文面は一般的な表現になりがちで、自分の言い回しや構成のクセが反映されない。**Gem**（ジェム）を使えば、自分のメールスタイルを事前に登録しておき、Gmail のサイドパネルからワンクリックで「自分らしい文面」を生成できる。

### Gem とは

Gem は、Gemini の回答スタイルや指示内容をあらかじめ登録しておけるカスタム AI 機能。一度作成すれば、Gmail のサイドパネルから呼び出して繰り返し使える。メールの文体、構成、敬語のルールなどを登録しておくことで、毎回指示しなくても自分のスタイルに沿った文面が生成される。

### 学外者向けと学内者向けで Gem を分ける

メールの文体は相手によって大きく異なるため、**学外者向けと学内者向けで Gem を分けて作成する**のが効果的。1つの Gem にすべてのパターンを詰め込むと指示が複雑になり、精度が下がりやすい。

| Gem 名 | 対象 | 文体の特徴 |
|--------|------|-----------|
| メール（学外） | 取引先、他大学、官公庁、保護者など | フォーマルな敬語、丁重な表現、時候の挨拶あり |
| メール（学内） | 教員、同僚、学生など | 丁寧だが簡潔、「お疲れさまです」で始まる |

さらに細かく分けたい場合は「教員向け」「学生向け」など追加してもよいが、まずは学外・学内の2つで始めるのがシンプル。

### 1. 過去の送信メールを収集する

Gmail の「送信済み」から、自分らしさがよく出ている代表的なメールを10〜20通選ぶ。

**選ぶ際のポイント:**
- 学外向け・学内向けそれぞれ5〜10通ずつ
- よく書く種類のメール（依頼、回答、案内、お礼、お詫びなど）
- 自分で「これはうまく書けた」と思うメール

### 2. Gemini でスタイルを分析する

収集したメールを [gemini.google.com](https://gemini.google.com/) に貼り付けて、スタイルを分析する。学外向けと学内向けで別々に分析する。

**手順:**

1. [gemini.google.com](https://gemini.google.com/) を開く
2. 学外向けメール5〜10通をまとめてコピーし、以下のプロンプトと一緒に貼り付ける

**プロンプト:**

```
以下は私が送信したメールです。
これらのメール文面を分析し、私のメールの書き方の特徴をまとめてください。

以下の観点で分析してください。
- メールの冒頭の書き出し
- 本題への入り方
- 締めの定型表現
- よく使う敬語表現・クッション言葉
- 避けている表現
- 1文の長さや箇条書きの使い方

Gemini の Gem に登録するカスタム指示として使える形式で
出力してください。

---
（ここに送信済みメールを貼り付け）
```

3. Gemini が出力したスタイルガイドを、次のステップで Gem のカスタム指示に使う
4. 学内向けメールでも同様に分析する

### 3. Gem を作成する

#### 学外者向け Gem の作成

1. [gemini.google.com](https://gemini.google.com/) の左メニューから **「Gem マネージャー」** をクリック
2. **「新しい Gem を作成」** をクリック
3. Gem の名前を入力（例: 「メール（学外）」）
4. カスタム指示欄に、前のステップで Gemini が分析したスタイルガイドを登録する

**カスタム指示の例（学外者向け）:**

```
あなたは私の代わりにメールの返信案や新規メールを作成するアシスタントです。
以下のスタイルに従って文面を作成してください。

【対象】学外の相手（取引先、他大学、官公庁、保護者など）

【構成】
- 宛名の後、時候や状況に応じた挨拶を1文入れる
- 「さて、」「このたびは」で本題に入る
- 詳細は箇条書きでまとめる
- 締め: 「何卒よろしくお願い申し上げます。」

【敬語】
- 丁重な敬語を使う（「いただけますでしょうか」「ご高配を賜り」など）
- 依頼には必ずクッション言葉を添える（「恐れ入りますが」「ご多忙のところ恐縮ですが」）

【避ける表現】
- 「させていただく」の多用
- カジュアルすぎる表現（「〜ですね」「〜しますね」）

【メール例】
（過去のメール例を2〜3通貼り付け）
```

5. **「保存」** をクリック

#### 学内者向け Gem の作成

同様の手順で学内向けの Gem を作成する。

**カスタム指示の例（学内者向け）:**

```
あなたは私の代わりにメールの返信案や新規メールを作成するアシスタントです。
以下のスタイルに従って文面を作成してください。

【対象】学内の相手（教員、同僚の職員、学生など）

【構成】
- 「○○先生」「○○さん」の後、「お疲れさまです。」で始める
- 「○○の件でご連絡いたしました。」で本題に入る
- 詳細は箇条書きでまとめる
- 締め: 「お忙しいところ恐れ入りますが、よろしくお願いいたします。」
- 返信不要の場合: 「特にご返信は不要です。」を添える

【敬語】
- 教員には「先生」、職員には「さん」で統一
- 「いただけますか」（「いただけますでしょうか」は使わない）
- 簡潔だが丁寧な表現を基本とする

【学生向けの場合】
- 丁寧語ベースでわかりやすく
- 手続きの手順は番号付きで具体的に示す
- 窓口の場所や受付時間など実用的な情報を含める

【避ける表現】
- 「させていただく」の多用
- 過度にフォーマルな表現（学内のやり取りには不自然）

【メール例】
（過去のメール例を2〜3通貼り付け）
```

### 4. Gmail のサイドパネルから Gem を使う

Gem を作成すると、Gmail のサイドパネルから直接呼び出して使える。

**手順:**

1. Gmail で返信したいメールを開く
2. 右上の **Gemini アイコン** をクリックしてサイドパネルを開く
3. サイドパネル上部の Gem リストから、作成した Gem（「メール（学外）」または「メール（学内）」）を選択
4. 指示を入力する

**入力例:**

```
このメールに対して、出席できる旨の返信を作成してください。
当日の資料は事前に確認しておく旨も添えてください。
```

5. Gem が開いているメールの内容を読み取り、登録したスタイルに沿った返信案を生成する
6. 生成された文面の下にある **「挿入」ボタン** をクリックすると、返信本文に自動挿入される
7. 内容を確認・微修正して送信

**ポイント:** Gem はメールの差出人名や本文を自動で読み取るため、「このメールに返信して」と指示するだけで、相手の名前や文脈を反映した返信案が生成される。

### 5. Gem を育てる

運用しながら Gem のカスタム指示を改善する。

- 生成された文面で「自分ならこう書かない」と感じた表現があれば、カスタム指示の「避ける表現」に追加する
- 新しいパターンのメール（お詫び、異動の挨拶など）を書いたら、メール例として追加する
- 半年〜1年に一度、指示全体を見直す

Gem のカスタム指示は [gemini.google.com](https://gemini.google.com/) の Gem マネージャーからいつでも編集できる。

## 導入効果

| 項目 | 導入前 | 導入後 |
|------|--------|--------|
| 返信メール作成時間 | 1通あたり 5〜15 分 | 1〜3 分（確認・修正のみ） |
| 新規メール作成時間 | 1通あたり 10〜20 分 | 2〜5 分 |
| AI 生成文の修正量 | 大幅に書き直し | Gem でスタイル統一、微修正で済む |
| 敬語・表現への不安 | 特に新任者に多い | Gem が適切な表現を提案 |
| 学外・学内の書き分け | 毎回意識して調整 | Gem を切り替えるだけ |

## 応用例

- **チーム共通の Gem**: 部署全体のメール文体を統一したい場合、共通の Gem を作成してチームで共有する
- **業務別の Gem**: 「入試関連メール」「経理関連メール」など、業務内容に応じた Gem を作成し、専門用語や定型表現を登録する
- **メール以外の文書**: 学内通知文や報告書の作成にも Gem を応用できる

## 注意点

- 生成された文面は必ず送信前に内容を確認する。事実関係（日時、金額、固有名詞）が正確かどうかは自分で検証する
- Gem のカスタム指示にメール例を含める場合、個人情報や機密情報は削除してから登録する
- Gem のスタイルはあくまで「傾向」であり、すべてのメールに画一的に適用するものではない。状況に応じて柔軟に調整する
