## 課題

海外の大学や機関から英語・中国語・韓国語などでメールが届くことがある。Gemini に「翻訳して返信も作って」と一括で依頼すると、翻訳と返信作成が一つのプロンプト内で混ざり合い、翻訳に元の指示文が紛れ込んだり、返信文の日本語訳に要約や意訳が入ったりすることがある。正確な翻訳と、正確な返信文を、それぞれ独立して確認したい。

## この記事が役立つケース

**導入が効果的な場合:**
- 外国語のメール対応が月に数件以上あり、翻訳と返信作成に時間がかかっている
- Gemini に一括で依頼しても、翻訳と返信作成が混在して精度に不満がある
- 返信文の外国語訳を正確に確認してから送りたい

**現状のままで十分な場合:**
- 外国語メールの対応頻度が極めて低い（年に数回以下）
- 担当者が外国語に堪能で、翻訳支援が不要

## 解決策

**Google AI Studio** の Build 機能を使い、外国語メール対応に特化したWebアプリをノーコードで作成する。アプリは以下の4つのステップを**独立した処理**として順番に実行する。

| ステップ | 処理内容 | 表示される結果 |
|---------|---------|-------------|
| 1 | 外国語メールの貼り付け | 受信メールの日本語訳 |
| 2 | 返信したい内容を日本語で入力 | — |
| 3 | 日本語の入力内容をもとに、相手の言語で返信文を作成 | 外国語の返信文 |
| 4 | 返信文の日本語訳を表示 | 返信文の日本語訳 |

各ステップが独立した API 呼び出しになるため、ステップ1の翻訳にステップ2以降の指示が混入しない。ステップ4の日本語訳もステップ2の入力内容に引きずられず、純粋に外国語の返信文だけを翻訳した結果が得られる。

## Gemini に一括で依頼する場合との違い

Gemini のチャットに「このメールを翻訳して、こう返信して、その返信の日本語訳も出して」と依頼すると、すべてが1つの会話コンテキストの中で処理される。このとき以下の問題が起きやすい。

- **翻訳への指示の混入** — 「返信を作って」という指示が翻訳結果に影響し、原文にない意訳や要約が入る
- **返信文の日本語訳が不正確** — 日本語で書いた返信意図が残っているため、実際の外国語返信文の直訳ではなく、入力した日本語をそのまま返してしまう
- **言語の混在** — 長い会話の中で言語の切り替えが不安定になる

このアプリでは各処理を独立した API 呼び出しにすることで、これらの問題を構造的に解消する。

## Google AI Studio とは

Google が提供する AI 開発環境。Gemini API を使ったアプリの試作・開発ができる。**Build 機能**を使えば、作りたいアプリの内容を日本語で説明するだけで、コードを書かずに動作するWebアプリが生成される。Google アカウントがあれば無料で利用できる。

## 前提条件

- Google アカウント（個人アカウントで利用可能）
- Google AI Studio へのアクセス（[aistudio.google.com](https://aistudio.google.com/)）

## 構築手順

### 1. Google AI Studio を開く

[aistudio.google.com](https://aistudio.google.com/) にアクセスし、Google アカウントでログインする。

### 2. Build モードを起動する

左側のメニューから **「Build」** をクリックする。アプリの説明を入力するチャット画面が表示される。

### 3. アプリの内容を指示する

以下のプロンプトを入力して送信する。

```
以下の仕様で外国語メール対応アプリを作成してください。

【アプリの概要】
外国語で届いたメールの翻訳、返信文の作成、返信文の確認を
ステップごとに行うアプリです。

【画面構成】
1つの画面に、以下の要素を上から順に配置してください。

■ ステップ1: 受信メールの翻訳
- テキストエリア: 外国語メールの本文を貼り付ける欄
  （ラベル: 「外国語メールを貼り付けてください」）
- 「翻訳する」ボタン
- 翻訳結果の表示エリア（ラベル: 「日本語訳」）

■ ステップ2: 返信内容の入力
- テキストエリア: 返信したい内容を日本語で入力する欄
  （ラベル: 「返信したい内容を日本語で入力してください」）
- 「返信文を作成する」ボタン

■ ステップ3: 外国語の返信文
- 返信文の表示エリア
  （ラベル: 「返信文（相手の言語）」）
- コピーボタン（返信文をクリップボードにコピー）

■ ステップ4: 返信文の日本語訳
- 返信文の日本語訳の表示エリア
  （ラベル: 「返信文の日本語訳（確認用）」）

【重要な処理ルール】
- ステップ1の翻訳、ステップ3の返信文作成、ステップ4の日本語訳は、
  それぞれ独立した API 呼び出しで処理すること。
  1つのプロンプトにまとめないこと。
- ステップ1では、貼り付けられたテキストの言語を自動判別し、
  日本語に翻訳する。翻訳以外の情報（要約、コメント等）は含めない。
- ステップ3では、ステップ1で判別した言語で返信文を作成する。
  ビジネスメールとして適切な文体にする。
  ステップ2の日本語入力の内容を正確に反映すること。
- ステップ4では、ステップ3で生成した外国語の返信文のみを
  日本語に翻訳する。ステップ2の入力内容は参照しないこと。
  純粋にステップ3の外国語テキストの直訳とすること。

【デザイン】
- 各ステップの間に区切り線を入れて視覚的に分離する
- ステップ番号を表示する
- 落ち着いた配色にする（白背景、青系のアクセントカラー）
- レスポンシブ対応（スマホでも使えること）
```

### 4. プレビューで動作を確認する

プロンプトを送信すると、画面右側にアプリのプレビューが表示される。以下の点を確認する。

**確認項目:**

- 外国語メールを貼り付けて「翻訳する」ボタンを押すと、日本語訳が表示されるか
- 日本語で返信内容を入力して「返信文を作成する」ボタンを押すと、外国語の返信文が表示されるか
- 返信文の日本語訳が、ステップ2の入力内容ではなく、ステップ3の外国語返信文の直訳になっているか
- コピーボタンが動作するか

**テスト用メール例（英語）:**

```
Dear Sir/Madam,

I am writing to inquire about the possibility of establishing
a student exchange program between our universities.
We are particularly interested in your Faculty of Engineering
and would like to discuss potential collaboration opportunities.

Could we arrange a meeting at your earliest convenience?

Best regards,
John Smith
International Office
University of Melbourne
```

### 5. 必要に応じて修正を指示する

動作に問題がある場合は、チャットパネルで修正を指示する。

**修正指示の例:**

```
ステップ4の日本語訳が、ステップ2で入力した日本語と
ほぼ同じ内容になっています。
ステップ3の外国語テキストだけを入力として翻訳するように
修正してください。ステップ2の内容は参照しないでください。
```

```
返信文の文体がカジュアルすぎます。
大学の公式な対外文書として適切な丁寧な表現に
なるように調整してください。
```

```
言語の自動判別がうまくいきません。
ステップ1のテキストエリアの上に、
言語を手動で選択するプルダウン
（英語 / 中国語 / 韓国語 / フランス語 / その他）
を追加してください。
```

### 6. アプリの共有

完成したアプリは以下の方法で共有できる。

- **共有リンク** — Google AI Studio の共有機能で URL を発行し、学内で共有する
- **Cloud Run にデプロイ** — 「Deploy」ボタンから Google Cloud Run にデプロイすると、独立した URL でアクセスできるWebアプリになる（Google Cloud のアカウントが必要）

学内での利用であれば、共有リンクで十分。

## 運用フロー

### 外国語メールを受信したとき

1. アプリを開く
2. 受信メールの本文をコピーして、ステップ1に貼り付ける
3. 「翻訳する」を押して日本語訳を確認する
4. ステップ2に、返信したい内容を日本語で入力する
5. 「返信文を作成する」を押す
6. ステップ3の外国語返信文と、ステップ4の日本語訳を突き合わせて確認する
7. 問題なければ、コピーボタンで返信文をコピーし、メールに貼り付けて送信する

## 導入効果

| 項目 | 導入前 | 導入後 |
|------|--------|--------|
| 外国語メールの内容把握 | 翻訳サイトで翻訳（翻訳精度にばらつき） | アプリ内で日本語訳を即確認 |
| 返信文の作成 | 翻訳サイトで日→外翻訳（不自然な表現になりがち） | ビジネス文体で自然な返信文を自動作成 |
| 返信文の内容確認 | 自分で再翻訳して確認（手間がかかる） | ステップ4で自動的に日本語訳が表示される |
| 翻訳の正確性 | 一括処理で指示内容が翻訳に混入することがある | 各ステップが独立しており、混入が発生しない |
| 1通あたりの対応時間 | 15〜30 分 | 5〜10 分 |

## 応用例

- **留学生対応**: 留学生からの母国語での問い合わせに、相手の言語で丁寧に返信する
- **海外の大学との事務連絡**: 協定校との交換留学・共同研究に関するメールのやり取り
- **多言語の公式文書作成**: 英語の案内文を作成し、日本語訳で内容を確認してから送付する

## 注意点

- AI が生成した翻訳・返信文は、送信前に必ず内容を確認する。特に日程、金額、固有名詞の正確性に注意する
- 機密情報を含むメールを扱う場合は、Google AI Studio の利用規約とデータの取り扱いポリシーを確認する。Google Workspace アカウントで利用する場合はエンタープライズ級のデータ保護が適用される
- Google AI Studio の Build 機能は無料で利用できるが、API の呼び出し回数には上限がある。大量のメール処理には向かない
- 自動判別が難しい言語（方言、混在テキスト等）の場合は、言語選択のプルダウンを追加して手動で指定する方が確実
