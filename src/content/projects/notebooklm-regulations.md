## 課題

法令や学内規則は文書量が膨大で、必要な条文を探すだけでも時間がかかる。複数の規則にまたがる内容を確認する場合はさらに手間が増え、解釈に迷うこともあった。

## この記事が役立つケース

**導入が効果的な場合:**
- 参照する法令・規則が多く、条文検索に時間がかかっている
- 複数の規則を横断して確認する業務が頻繁にある
- 新任者が規則を理解するまでに時間がかかっている

**現状のままで十分な場合:**
- 参照する規則が少数で、担当者が内容を把握している
- 規則の確認頻度が低く、年に数回程度

## 解決策

**Google NotebookLM** に法令・規則の PDF を読み込ませ、対話形式で必要な情報を即座に引き出せる「規則チャットボット」を構築した。

## NotebookLM とは

Google が提供する AI ノートブックツール。PDF や Google ドキュメントなどを「ソース」としてアップロードすると、その内容に基づいて質問への回答、要約、比較などを行える。回答には必ず参照元（出典）が明示されるため、根拠を確認しやすい。

## 構築手順

### 1. ソース資料の準備

確認したい法令・規則を PDF 形式で用意する。

**法令の場合:**
- [e-Gov 法令検索](https://elaws.e-gov.go.jp/) にアクセス
- 対象の法令を検索し、PDF でダウンロード

**学内規則の場合:**
- 学内ポータルやイントラネットから規則集を PDF でダウンロード
- 複数ファイルに分かれている場合はそのまま複数追加可能

### 2. NotebookLM にアップロード

1. [NotebookLM](https://notebooklm.google.com/) にアクセス
2. 「新しいノートブック」を作成
3. 「ソースを追加」から PDF をアップロード

1つのノートブックに複数のソースを追加できる。関連する法令・規則をまとめて追加しておくと、横断的な質問が可能になる。

### 3. 対話形式で質問する

アップロードが完了すれば準備完了。チャット欄に質問を入力するだけで、ソース資料に基づいた回答が得られる。

## 活用例

### 特定の条文を探す

```
出張旅費の日当について定めている条文はどれですか？
```

→ 該当する規則名・条項番号とともに、日当の金額や条件が回答される。

### 複数の規則を横断して確認する

```
教員の兼業に関して、就業規則と兼業規程の
両方に定めがありますか？矛盾する内容はありませんか？
```

→ 両方の規則から関連条文を抽出し、整合性を確認した結果が回答される。

### 規則の内容をわかりやすく説明する

```
個人情報保護に関する規程の要点を、
新入職員向けにわかりやすく箇条書きで説明してください。
```

→ 専門用語を平易な言葉に置き換えた説明が生成される。

### 具体的なケースについて判断材料を得る

```
職員が副業として週末にオンライン講座の講師を
する場合、届出は必要ですか？
関連する規則の条文を示してください。
```

→ 兼業規程の該当条文を引用しながら、届出の要否が回答される。

### 法改正の影響を確認する

```
2024年4月施行の労働条件明示ルールの変更点は何ですか？
現在の就業規則で対応が必要な箇所はありますか？
```

→ 改正内容とソース資料を照合し、見直しが必要な条文を指摘してくれる。

## 導入効果

| 項目 | 導入前 | 導入後 |
|------|--------|--------|
| 条文の検索時間 | 10〜30 分 | 数秒〜1 分 |
| 複数規則の横断確認 | 各規則を個別に開いて比較 | 1つの質問で横断回答 |
| 規則の解釈・説明 | 担当者の経験に依存 | AI が条文根拠つきで説明 |
| 新任者の規則理解 | マニュアル読み込みに数日 | 対話形式で即座に確認 |

## 効果的に使うコツ

- **ソースは目的別にノートブックを分ける** — 「人事関連規則」「経理関連規則」のようにテーマごとにノートブックを作ると、回答精度が上がる
- **質問は具体的に** — 「旅費規程について教えて」より「国内出張の日当の金額と支給条件を教えて」の方が的確な回答が得られる
- **回答の出典を必ず確認する** — NotebookLM は参照元を明示するので、原文に立ち返って確認する習慣をつける
- **音声出力を活用する** — 長い規則の概要を音声で聴くことで、移動時間を活用した効率的なインプットが可能

## 注意点

- NotebookLM の回答はソース資料に基づくが、法的な最終判断は専門家に確認すべき
- 法改正があった場合、ソース PDF を差し替える必要がある。定期的な更新を忘れないこと
- ソースの量が多すぎると回答精度が下がる場合がある。目的に応じてノートブックを分割するのが望ましい
- 機密性の高い内部規則をアップロードする場合は、組織のセキュリティポリシーを確認すること
