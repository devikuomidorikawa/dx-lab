## 課題

大学事務では、教員、学生、外部機関など立場の異なる相手に日常的にメールを送信する。敬語の使い分けや文体の統一に気を使う必要があるが、忙しいときほどミスが起きやすい。特に新任職員は適切な敬語に不安を感じることが多かった。

## この記事が役立つケース

**導入が効果的な場合:**
- 立場の異なる相手（教員・学生・外部機関）へのメールが多く、敬語の使い分けに不安がある
- 新任職員がメール作成に時間をかけている
- メールの品質にばらつきがあり、統一したい

**現状のままで十分な場合:**
- メールの送信頻度が低く、上司や同僚に確認する余裕がある
- 十分な経験があり、敬語の使い分けに迷うことがない

## 解決策

**Gemini in Gmail** の「文章を洗練」機能や、Google ドキュメントの Gemini を使い、送信前にメール文面を校正する。敬語の誤り、冗長な表現、わかりにくい箇所を指摘し、改善案を提示する。

## 活用方法

### 1. Gmail の Gemini 機能を使う

Gmail のメール作成画面で Gemini アイコンをクリックし、以下の機能を活用する。

- **「文章を洗練」**: 文体をフォーマルに調整
- **「短くする」**: 冗長な表現を簡潔に
- **「詳しくする」**: 説明が不足している部分を補完

### 2. より詳細な校正を行う場合

メール本文を Google ドキュメントに貼り付け、Gemini サイドパネルで詳細な校正を依頼する。

**プロンプト例:**

```
以下のメール文面を校正してください。

【チェック項目】
1. 敬語の誤り（尊敬語・謙譲語・丁寧語の使い分け）
2. 二重敬語になっていないか
3. 主語と述語のねじれ
4. 冗長な表現
5. わかりにくい箇所
6. 誤字・脱字

修正箇所は【修正前】→【修正後】の形式で示し、
修正理由も簡潔に説明してください。

【メール文面】
{{メール本文}}
```

### 3. 相手に応じた文体調整

**教員宛の場合:**

```
このメールを教員宛の文面として適切な敬語に調整してください。
「先生」の呼称を使い、依頼は柔らかい表現にしてください。
```

**外部機関宛の場合:**

```
このメールを学外の取引先に送る文面として
フォーマルな敬語に調整してください。
時候の挨拶は不要ですが、丁重な表現にしてください。
```

**学生宛の場合:**

```
このメールを学生向けの文面に調整してください。
丁寧語を基本としつつ、わかりやすい表現にしてください。
専門用語には簡潔な説明を添えてください。
```

## 導入効果

| 項目 | 導入前 | 導入後 |
|------|--------|--------|
| メール作成の不安 | 敬語に自信がない（特に新任） | AI がチェックしてくれる安心感 |
| 確認の手間 | 上司に確認を依頼 | 自分で一次チェック可能 |
| 文面の品質 | 担当者によりばらつき | 一定の品質を確保 |

## 応用例

- **通知文書の校正**: 学内通知や依頼文の文面チェック
- **英語メールの作成補助**: 海外の大学や機関への英語メールのドラフト作成・校正
- **議事録の文体統一**: 複数人で作成した議事録の文体を統一

## 注意点

- Gemini の修正提案は参考として活用し、最終判断は送信者が行う
- 機密情報を含むメールの内容を Gemini に入力する際は、組織のセキュリティポリシーに従う
- 形式的すぎる敬語よりも、相手との関係性に応じた自然な表現を優先する
