## 課題

部署の共有ドライブにファイルが無秩序に保存され、目的のファイルを探すのに時間がかかっていた。命名規則がなく、「最終版」「最終版2」「本当の最終版」のようなファイルが乱立。担当者以外はどこに何があるかわからない状態だった。

## 解決策

**Gemini** を使い、現状のフォルダ構成を分析して最適なフォルダ構成案と命名規則を作成。整理作業の手順書も自動生成し、部署全体で共有フォルダの運用ルールを標準化した。

## 構築手順

### 1. 現状のフォルダ構成を把握

Google ドライブの共有フォルダの現状を整理する。フォルダ名とファイル数をリストアップし、Gemini に分析を依頼する。

**プロンプト例:**

```
以下は部署の共有ドライブのフォルダ構成です。
この構成の問題点を指摘し、改善案を提案してください。

【現状のフォルダ構成】
総務課共有/
  ├── 会議資料/
  ├── 2024年度/
  ├── 田中さんフォルダ/
  ├── 古いファイル/
  ├── コピー/
  ├── 雑多/
  └── 新しいフォルダ/
```

### 2. フォルダ構成の設計

Gemini が提案する構成案をもとに、部署に最適なフォルダ構成を決定する。

**推奨構成例:**

```
総務課共有/
  ├── 01_会議・委員会/
  │   ├── 教授会/
  │   ├── 運営会議/
  │   └── 各種委員会/
  ├── 02_規程・マニュアル/
  ├── 03_申請・届出/
  ├── 04_予算・経理/
  ├── 05_人事・労務/
  ├── 06_広報・イベント/
  ├── 07_テンプレート/
  └── 99_アーカイブ/
      ├── 2023年度/
      └── 2024年度/
```

### 3. 命名規則の策定

Gemini に命名規則の案を生成させる。

**プロンプト例:**

```
大学事務の共有ドライブで使うファイル命名規則を作成してください。

【要件】
- 検索しやすいこと
- 日付がわかること
- バージョン管理ができること
- 日本語でわかりやすいこと

具体例も5つ以上示してください。
```

**命名規則の例:**

```
[日付]_[種別]_[内容]_[バージョン]

例:
20250305_議事録_教授会.docx
20250301_稟議_PC購入_v2.docx
20250228_報告書_FD活動_最終.pdf
```

### 4. 運用ガイドの共有

作成したフォルダ構成と命名規則を Google ドキュメントにまとめ、部署メンバーに共有する。

## 導入効果

| 項目 | 導入前 | 導入後 |
|------|--------|--------|
| ファイル検索時間 | 5〜15 分 | 1〜2 分 |
| ファイルの重複 | 頻繁に発生 | 命名規則で防止 |
| 新任者の理解 | 構成がわからない | ガイドで即理解 |

## 注意点

- 既存ファイルの移動は一度にやらず、段階的に進める
- 命名規則は複雑にしすぎない。守られなければ意味がない
- 定期的に不要ファイルをアーカイブフォルダに移動するルールを設ける
